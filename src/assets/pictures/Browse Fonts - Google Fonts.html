Use on the web

To embed a font, copy the code into the <head> of your html

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Alfa+Slab+One&family=Anton&family=Bungee&family=Bungee+Inline&family=Bungee+Shade&display=swap" rel="stylesheet">

CSS rules to specify families

  1. font-family: 'Alfa Slab One', cursive;
  1. font-family: 'Anton', sans-serif;
  1. font-family: 'Bungee', cursive;
  1. font-family: 'Bungee Inline', cursive;
  1. font-family: 'Bungee Shade', cursive;
